HTML Cheat Sheet!

Table Tags

Tag Name Description
<table> Table This should wrap around your entire html table.
<thead> Table Head This should be used to designate the names of the rows
<tbody> Table Body This should be used to designate where the actual data will go.
<tr> Table Row This will designate a new row in your table.
<td> Table Data This will be where the actual data goes inside of your table!
<tfoot> Table Footer This will be for putting information at the bottom of the table.

Table Attributes

These tags go inside the table tags (i.e. <td colspan="2">) in order for them to work.

Attribute Name Description
colspan Column Span Defines how many columns a td (table data) element should span.
rowspan Row Span Defines how many rows a td (table data) element should span.
